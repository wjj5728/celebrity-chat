# celebrity-chat

v0.1.0 本地原型：
- 选择名人
- 输入问题
- 以名人视角回答
- 显示本轮回答依据（基于名人生平事实）

v0.2.0 迭代：
- 新增 API 抽象层（`src/api.js`）
- 聊天请求改为异步调用（模拟后端）
- 新增模型选择器（可切换模型名）
- 消息中显示本轮调用模型

v0.3.0 迭代：
- 新增轻量 RAG 层（`src/rag.js`）
- 按问题关键词检索人物事实块
- 回答使用“依据优先”草案
- 右侧展示可追溯来源信息

v0.4.0 迭代：
- 新增安全策略模块（`src/safety.js`）
- 高风险问题阻断（block）
- 中风险问题追加安全提示（warn）
- 聊天流程接入审核判断

## Run
直接用浏览器打开 `index.html` 即可。

- v0.6.0: 角色卡后台管理与评测面板
